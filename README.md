# Economic News Sentiment and Market Trend Prediction Using FinBERT

## Project Overview
This project investigates whether financial news sentiment can be used to understand and predict stock market trends. Transformer-based sentiment classification using FinBERT is combined with machine learning models for market prediction.

## Objectives
- Classify financial news into positive, neutral, and negative sentiment categories
- Compare transformer-based NLP with traditional NLP baselines
- Integrate sentiment signals with historical market indicators
- Predict short-term market movement using machine learning models

## Technologies Used
- Python
- FinBERT
- Hugging Face Transformers
- Random Forest
- Logistic Regression
- Scikit-learn
- Pandas
- Matplotlib

## Methodology
1. Collect and preprocess financial news headlines
2. Perform sentiment classification using FinBERT
3. Merge sentiment scores with stock market indicators
4. Train machine learning models for market trend prediction
5. Evaluate prediction accuracy and feature importance

## Models Used
- FinBERT
- TF-IDF Baseline
- Logistic Regression
- Random Forest

## Key Features
- Transformer-based financial NLP
- Sentiment-aware market prediction
- Feature importance analysis
- Financial data integration

## Applications
This project demonstrates how NLP and machine learning can support:
- Financial analytics
- Market forecasting
- Sentiment-driven investment research
- AI applications in finance
